# Seaborn Data Visualization Examples

This notebook serves as a comprehensive guide and demonstration for various data visualization techniques using the Seaborn library in Python. It showcases different plot types, color palettes, and data exploration methods through practical examples.

## Datasets Used
- **Tips Dataset**: Used to demonstrate various categorical and relational plots.
- **Titanic Dataset**: Explored for distribution, relationship, and categorical insights.
- **Loan Data (from CSV)**: Used for custom data loading and basic plotting.

## Key Visualizations & Concepts Covered
- **Basic EDA**: `head()`, `tail()`, `describe()`, `dtypes`, `isnull()`, `value_counts()`
- **Color Palettes**: Demonstrating `sns.color_palette()` and `sns.palplot()`.
- **Distribution Plots**: `distplot` (now `histplot` or `displot`), `violinplot`.
- **Relational Plots**: `jointplot`, `scatterplot`, `lineplot`.
- **Categorical Plots**: `stripplot`, `swarmplot`, `barplot`, `pointplot`.
- **Matrix Plots**: `heatmap`, `clustermap`.
- **Multi-Plot Grids**: `pairplot`, `FacetGrid` for complex conditional visualizations.

## Libraries Used
- `pandas` for data manipulation.
- `numpy` for numerical operations.
- `matplotlib.pyplot` for plot customization.
- `seaborn` for high-level statistical data visualization.

## How to Use
Run the cells sequentially to see the data loading, exploration, and various visualization examples in action.
